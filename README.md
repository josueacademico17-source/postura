# Inspector Postural

Base profesional Three.js/Vite para inspeccion ergonomica en una oficina 3D.

## Ejecutar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Controles

- `W`, `A`, `S`, `D`: movimiento FPS suave.
- `Shift`: caminar rapido.
- `Q`: activar/desactivar escaner postural.
- `E`: hablar/inspeccionar empleado enfocado.
- `B`: abrir tienda ergonomica.
- `T`: mostrar reporte rapido.
- `Esc`: cerrar panel o pausar.

## Estructura

El proyecto esta separado por sistemas: nucleo, player, scanner, npc, office, rendering, ui, data y utils. Los assets tienen carpetas preparadas para modelos, texturas, sonidos y logos.
